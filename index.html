Ebenezer

v0.1 "Tiny Tim"
A command line personal accounting tool written in Python.

Why another accounting tool ?

Ebenezer spawned from a recurring problem of mine : I'm bad at keeping track of my money. I get by, but I could do much more saving if I was a little more consistent with my accounting. So I've been around, and I tried a bunch of system. On paper, with a spreadsheet, with dedicated software, and nothing really cut it. So I decided to set out and create my own tool. The driving principle behind it is that I want it as simple as possible, but still with a couple of specialised features. It is currently only command-line, because I find it faster than mouse-heavy interfaces, and because I plan on making it on a client-server model. I've also been really impressed by some Android applications that let you keep track of payments withing groups of people, so I want to include that too. Eventually, I plan on having a very versatile backend that you will be able to manipulate via a variety of interfaces which can hide most of the complexity.

Current status

The current development status will be documented on the wiki.
The current version is the first public release of Ebenezer, codenamed "Tiny Tim" (of if you prefer versions number : 0.1). It can do a few simple things, such as storing transactions on any number of accounts, and associate transactions to a person other than you. Ie. if I associate an outbound transaction, say -100€, to my friend "Tim", it will mean that Tim owes me 100€, and that will be reported in the application.
